# Library Management System (LMS)

A **Library Management System** built using **JavaServer Faces (JSF)**, **Hibernate**, **MySQL**, and deployed on **WildFly**. This application allows librarians to manage books, members, and transactions efficiently, including issuing and returning books, tracking overdue items, and generating reports.

---

## Features

- **Dashboard**: Quick access to all functionalities.
- **Book Management**:
  - Add new books to the library.
  - Track available copies.
- **Member Management**:
  - Register new library members.
  - Manage member details.
- **Issue & Return Books**:
  - Issue books to members with due dates.
  - Process book returns and update inventory.
- **Reports & Analytics**:
  - View overdue items.
  - Track active circulation.

---

## Technologies Used

- **Frontend**: JavaServer Faces (JSF), Bootstrap 5, Font Awesome
- **Backend**: Java EE, Hibernate (ORM)
- **Database**: MySQL
- **Server**: WildFly
- **Build Tool**: Maven
